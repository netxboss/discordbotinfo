📜 Crypto News Bot – Privacy Policy

Effective Date: June 15, 2025
Last Updated: June 15, 2025

1. 🤖 About the Bot

Crypto News Bot is a Discord bot that provides real-time cryptocurrency news, price lookups, price alerts, and chart visualizations. It uses external APIs like CoinGecko and CoinDesk to fetch up-to-date data.

2. 🔐 Data We Collect

The bot collects only minimal data necessary for its features:

  • User ID – for tracking price alerts
  • Coin input queries – e.g., symbol or name, used for lookups
  • Price targets – for alerting purposes

We do not collect: personal info, message contents, IPs, or emails.

3. 🧠 How Data is Used

Data is only used to:

  • Respond to slash commands (like /price or /alert)
  • Send requested news or crypto data
  • Notify users of triggered alerts

No data is sold or shared with third parties.

4. ☁️ Data Storage

User data (alerts) is currently stored in memory only, meaning:

  • It is temporary
  • It is cleared when the bot restarts
  • No database or file storage is used

5. ✅ User Control

Users can fully control their data:

  • /alert add – add an alert
  • /alert list – view all active alerts
  • /alert delete – remove a specific alert

6. 🔒 Security

All API requests are made over HTTPS, and no sensitive data is requested or stored. Reasonable efforts are made to ensure bot safety and reliability.

7. 📩 Contact

Contact the server admin or bot developer via Discord for any privacy-related questions.

8. 📘 Changes to the Policy

This policy may be updated in the future. Any updates will be shared in the bot’s code repository or via changelogs in your Discord server.
